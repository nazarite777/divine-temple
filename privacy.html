4. Data Storage and Security

4.1 Where We Store Your Data

Your data is stored on secure Firebase Cloud Firestore servers, which are hosted on Google Cloud Platform infrastructure. Firebase maintains multiple data centers worldwide and implements industry-standard security measures.

4.2 Security Measures

  • Encryption: Passwords are encrypted using Firebase's secure authentication system (Argon2 or bcrypt)
  • HTTPS: All data transmitted between your device and our servers is encrypted via SSL/TLS
  • Access Controls: Strict Firebase security rules limit who can access what data
  • Session Management: Secure session tokens with automatic expiration
  • Regular Monitoring: Automated security monitoring and logging

4.3 Data Retention

We retain your personal information for as long as your account is active or as needed to provide services. If you wish to delete your account, please contact us at edenconsciousnesssdt@gmail.com.
