```
Headlines: Bold sans-serif, 28-36px
Subheads: Medium weight, 18-24px
Body: Regular weight, 16px, 1.5 line-height
```

```
24px base grid

Padding: Same
Mobile: 16px padding, 100% width buttons
```

```
Buttons: Rounded, gold primary, purple outline secondary
Cards: Thin gold border, slight transparency, glow on hover
Icons: 32-48px, colorful (not monochrome)
Animations: 0.3s ease-in-out on all transitions
```
