## Usage Recommendations

### Page Placements:
1. **Header Logo** - Use `edenlogotransparent.png` (60x60px)
2. **Hero Section** - Use `logopathtotree.png` or featured image
3. **About Pages** - Use author/Nazir profile image (placeholder path: `/images/branding/nazir-profile.jpg`)
4. **Premium Pages** - Use smaller logo in sidebar (40x40px)
5. **Footer** - Use `edenlogotransparent.png` (30x30px)

### Color Recommendations:
- Primary Gold: #D4AF37
- Deep Purple: #2D1B4E
- Text on Logo: White
